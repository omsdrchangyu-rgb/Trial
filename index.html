設定雲端同步

建立 Firebase 專案，讓資料跨裝置即時同步

1

建立 Firebase 專案

前往 console.firebase.google.com，以 Google 帳號登入，點擊「新增專案」完成建立。

2

啟用 Firestore 資料庫

左側選單 →「Firestore Database」→「建立資料庫」→「以測試模式啟動」→ 選擇地區（建議 asia-east1）。

3

新增 Web 應用程式並取得設定

專案概覽頁點擊「</>」→ 完成後複製 firebaseConfig 內的 JSON 內容：

apiKey: "AIzaSy...", authDomain: "your-app.firebaseapp.com", projectId: "your-app", storageBucket: "your-app.appspot.com", messagingSenderId: "...", appId: "..."
4

貼上設定並連線
